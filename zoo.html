    Rust in embedded systems

    Rust is a programming language that may be very hard to adapt as the lingua franca in embedded systems programming due to its large scope. The C programming language, while it doesn't offer nearly as many features to aid in ergonomics and safety, is much simpler to implement by a one-man team.

    Indeed, there may be value in making a "lighter" Rust, perhaps adopting a linear type system.
